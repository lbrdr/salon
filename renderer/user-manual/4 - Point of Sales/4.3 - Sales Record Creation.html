4.3 - Sales Record Creation

The sales record creation screen displays the necessary input fields needed to be answered when making a sales record such as customer name, preferred staff, services availed, and the payment.

The sales record creation screen screen
  1. Customer Field - The user will enter the new customer's name here.
  2. Preferred Staff Field - The user can select the customer's preferred staff if there is any.
  3. Transaction Date - It would display the time and date of the sales record creation/
  4. Services Availed - The user can select the services availed by the customer.
  5. Payment Field - The user will enter the amount paid by the customer
