4.1 - Point of Sales

The point of sales page is where the users would see sales records and its corresponding information. The sales records table is provided to show the sales records created in the system and its details such as the sales record ID, customer name, the servicing staff, and the date of when the transaction happened.

Buttons are provided in this page that would connect the user to other functions such as the sales record creation, sales record edit (admin only), and services edit (admin only). The sales record table would also allow the user to view more details about the sales record by clicking the view button in the deatils column. An admin may also select a sales record that would be used in the sales record edit.

The point of sales page
  1. Create Sales Record Button - This button will bring the user to the sales record creation screen wherein they can create a new sales record.
  2. Sales Record Table - In this table, the user may see the sales record created in the system. After clicking the view button in the details column, the user will be able to see more details about the sales record.
  3. Edit Sales Record Button (Admin-only) - Clicking this button allows the admin to correct and edit sales record information.
  4. Edit Services Button (Admin-only) - The admin can click this button to proceed to the services edit screen wherein they can change the services offered in sales records.
