4.4 - Sales Record Edit (Admin Only)

The Admin is able to change and edit sales record information. The user would only have to input the customer id and it would automatically fill out the information, the displayed information can now be edited and changed.

The sales record edit screen
  1. Customer ID - The user will input the unique identifier of the customer being edited to check if it exists.
  2. Customer's Fullname Field - The user will enter the customer's username here.
  3. Preferred Staff - The user is able to change the selected customer's preferred staff.
  4. Date of Transaction - This field automatically captures the date when the inventory record is being created.
  5. Services Availed - The user can change the selected services availed by the customer.
  6. Payment Field - The user will enter the amount paid by the customer.
