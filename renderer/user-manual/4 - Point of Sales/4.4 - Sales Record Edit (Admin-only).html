4.4 - Sales Record Edit (Admin Only)

The admin is able to change and edit sales record information in the sales record edit screen. In this screen, the admin must input the sales record ID of the sales record to be edited, and the details of the sales record to be updated. If a sales record was selected in the sales record table before clicking the edit sales record button, the fields will be pre-filled with the selected sales record's information.

The sales record edit screen
  1. Sales Record ID Field - The admin must input the sales record ID here.
  2. Check ID Button - Upon clicking this button, the entered sales record ID will be checked, and the inputs will be filled out by the information of the sales record with the entered sales record ID.
  3. Customer Field - The admin may input the serviced customer of the sales record here. This can be left blank if the customer is not registered.
  4. Preferred Staff Field - The servicing staff for the sales record is selected here.
  5. Transaction Date Field - The date of when the transaction happened is set here.
  6. Offered Services Input - The user can edit the services offered within the sales record here.
  7. Total Price - After editing the offered services, the total price of services will be displayed here.
  8. Payment Field - The admin must input the payment paid by the customer here.
  9. Change - After inputting the offered services and the payment, the amount of change received by the customer will be displayed here.
  10. Submit Button - By clicking this button, the sales record will be updated with the given information if all inputs are valid.
