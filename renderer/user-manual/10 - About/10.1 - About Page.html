10.1 - About Page

The About Page provides an overview of the system, including its version details, developers involved, and contact information for support.

The about screen
  1. System Overview - This section offers a brief explanation of the system's functionality and purpose, highlighting its main features and benefits for the users.
  2. Version - This section provides the version number of the system, detailing the current release and any notable updates or changes included in this version.
  3. Developers - This section provides information about the development team ensuring recognition and accountability.
  4. Contact Information - This section provides essential contact details of the developers for support, ensuring users know where to seek help if needed.
