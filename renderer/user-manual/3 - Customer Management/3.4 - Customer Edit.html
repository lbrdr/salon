3.4 - Customer Edit

In the customer edit screen, the user is able to change and edit customer information. To edit a customer's information, the user would have to input the customer ID. When an ID is entered and the check button is pressed, the provided fields will be filled with the information of the customer with the entered ID. If a customer was selected in the registered customers table in the customer management page (or the searched customers table in the customer search screen) when clicking the edit customer button, the fields in the customer edit screen would be automatically filled out with the selected customer's information.

The customer edit screen
  1. Customer ID Field - The user will enter the customer's id here.
  2. Check ID Button - Upon clicking this button, the entered customer ID will be checked and the fields will be filled out by information about the customer with the entered ID.
  3. Full Name Field - The user will enter the customer's full name here.
  4. Contact Field - The user will enter the customer's contact information here.
  5. Date of Registration - When editing a customer, the date of the customer's registration would be displayed here and cannot be edited.
  6. Preferred Staff Selection - Here, the user can select the customer's preferred staff if there is any.
  7. Submit Button - By clicking this button, the customer will be updated with the given information if all inputs are valid.
