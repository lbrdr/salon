3.3 - Customer Registration

The login screen is the first screen to be encountered when opening the program. Here, the user can be verified before using the system.

The customer registration screen
  1. Customer's Fullname Field - The user will enter the customer's username here.
  2. Contact Field - The user will enter the customer's contact here.
  3. Date of Registration - When registering a customer, the date would be automatically displayed.
  4. Preferred Staff - The user can select the customer's preferred staff if there is any
