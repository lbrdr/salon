3.3 - Customer Registration

The customer registration screen allows the user to register new customers into the system. To do so, they would input the information about the customer in the provided fields.

The customer registration screen
  1. Full Name Field - The user will enter the customer's full name here.
  2. Contact Field - The user will enter the customer's contact information here.
  3. Date of Registration - When registering a customer, the date would be automatically displayed.
  4. Preferred Staff Selection - Here, the user can select the customer's preferred staff if there is any.
  5. Submit Button - By clicking this button, a customer with the given information will be registered if all inputs are valid.
