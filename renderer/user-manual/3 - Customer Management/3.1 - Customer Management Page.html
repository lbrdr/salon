3.1 - Customer Management Page

The customer management page is the first screen the users would see when they have successfully logged in. The customer page includes a table of customers and their corresponding information such as the customer ID, their full name, their contact details, their preffered staff, and their date of registration.

Here, the user may click buttons that would connect you to other functions such as the customer registration, customer search, and customer edit. The registered customers table also allows the user to select a customer in the table (to be automatically filled in when the proceeding to the customer edit function).

The customer management page
  1. Register Customer Button - This button connects the user to the customer registration screen that allows them to register new customers.
  2. Registered Customers Table - In this table, the user can see the registered customers. By clicking the view button in the services column, the user will be able to see the customer-related services screen, showing the services availed by the customer and when. A user may also select a customer to be pre-filled when entering the customer edit page.
  3. Customer Search Button - This button connects to the Customer Search screen that allows them to search for customers.
  4. Edit Customer Button - Clicking this button allows the user to edit information about a registered customer. When a customer is selected in the customer table when clicking this button, the selected customer's information will be automatically filled in in the Customer Edit screen.
