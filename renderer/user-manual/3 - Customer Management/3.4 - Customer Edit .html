3.4 - Customer Edit (Admin Only)

The Customer Edit page allows the user to change and edit customer information. The user would only have to input the customer id and it would automatically fill out the informatiom.

The customer edit screen
  1. Customer ID - The user will input the unique identifier of the customer being edited to check if it exists.
  2. Customer's Fullname Field - The user will enter the customer's username here.
  3. Contact Field - The user will enter the customer's contact here.
  4. Date of Registration - This field automatically captures the date when the inventory record is being created.
  5. Preferred Staff - The user is able to change the selected customer's preferred staff.
