2.1 - Page Selector

When successfully logged in, user would be able to select the desired page they wanted to go to. The pages that would be displayed would be based on the user type (Admin and Staff). Pages visible to users are Customer Management, Point of Sales, Inventory, Help and About. Additional pages is added if user is Admin, pages added are User Management , Reports, and Backup.

  1. Staff side - The staff can only select from Customer Management, Point of Sales, Inventory, Help and About pages.
    2. The Staff side screen
  2. Admin side - All pages from the Staff would also be visible in the Admin side. Additional pages such as Report, User Management and Backup are added.
