5.2 - Inventory Item-Related Records View

The Inventory Item-Related Records View section provides two primary ways for users to view and manage their inventory records.

View inventory by records screen
View inventory by items screen
  1. View by Records - This button displays a list of all inventory records, showing details like item name, recording staff, and date recorded.
  2. View by Items - This button displays a list of all individual items that are unique, showing details like item name, current stock, and status.
