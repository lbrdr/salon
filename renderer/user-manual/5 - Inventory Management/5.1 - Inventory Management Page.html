5.1 - Inventory Management Page

The Inventory Management Page is where users can view and manage the salon's inventory. This page displays a list of all inventory items, their current stock levels, and related details.

The inventory management screen
  1. Create Inventory Record Button - The user can click this button to proceed to create inventory record procedure.
  2. View Records/ Items Button - This button allows users to manage how they want to see the inventory records.
  3. Edit Inventory Record Button - The user can click this button to proceed to modifying existing inventory record procedure.
