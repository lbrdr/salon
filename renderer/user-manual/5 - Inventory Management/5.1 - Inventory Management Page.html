5.1 - Inventory Management Page

The inventory management page is where users can view and manage the salon's inventory. This page may display a table of all inventory records and related details or a table of all inventory items, their current stock levels, and related details.

The inventory management screen
  1. Create Inventory Record Button - This button allows users to add new items to the inventory.
  2. View Records/Items Buttons - This button allows users to manage how they want to see the inventory records.
  3. Edit Inventory Record Button (Admin-only) - This button allows users to modify existing inventory records.

The inventory management page provides two primary ways for users to view the system's inventory details.

View inventory by records screen
View inventory by items screen
  1. View by Records - This button displays a list of all inventory records, showing details like item name, recording staff, and date recorded.
  2. View by Items - This button displays a list of all individual items that are unique according to the item name, item manufacturer, and unit. Each item will then be shown with its current stock and status, and a button is provided in the records column that will allow the user to view the inventory records related to the item.
