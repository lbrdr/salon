5.3 - Inventory Record Creation

The Inventory Record Creation Page allows users to add new inventory items to the system. This process ensures that all relevant details about the inventory items are accurately recorded for future reference and management.

The inventory record creation screen
  1. Item Name Field - The user will input the name of the inventory item here.
  2. Item Manufacturer Field - The user will enter the name of the manufacturer of the inventory item here.
  3. Unit of Measurement Field - The user will specify the unit of measurement for the inventory item (e.g., number of bottles) here.
  4. Cost per Unit Field - The user will input the cost of a single unit of the inventory item here.
  5. Quantity/Amount Field - The user will enter the total quantity/amount of the consumed or restocked inventory item here. Before entering the exact amount, the sign must be selected where a positive sign (+) represents a restock, and a negative sign (-) represents a consumption of the item.
  6. Recording Staff Field - This field records the ID and name of the staff member who is adding the inventory record. This is set to the current user by default and is only editable by an admin.
  7. Record Date Field - This field automatically captures the date when the inventory record is being created. This is only editable by an admin.
  8. View Changes Button - After inputting the needed details above, the user can click this button to view any changes that will be made to the inventory records and inventory items.
  9. Submit Button - By clicking this button, a new inventory record will be created with the given information if all inputs are valid.
