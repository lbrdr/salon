5.4 - Inventory Record Edit (Admin-only)

The Inventory Record Edit Page allows administrators to manage and edit inventory records.

The edit inventory record screen
  1. Inventory Record ID - The user will input the unique identifier of the inventory record being edited to check if it exists.
  2. Check ID Button - After clicking this button, the inventory record ID will be checked, and the input fields will be filled out by the information of the inventory record with the given inventory record ID.
  3. Item Name Field - The user will input the name of the inventory item here.
  4. Item Manufacturer Field - The user will enter the name of the manufacturer of the inventory item here.
  5. Unit of Measurement Field - The user will specify the unit of measurement for the inventory item (e.g., number of bottles) here.
  6. Cost per Unit Field - The user will input the cost of a single unit of the inventory item here.
  7. Quantity/Amount Field - The user will enter the total quantity/amount of the consumed or restocked inventory item here.
  8. Recording Staff Field - This field records the ID and name of the staff member who created the inventory record.
  9. Record Date Field - This field automatically captures the date when the inventory record is being created.
  10. View Changes Button - After inputting the needed details above, the user can click this button to view any changes that will be made to the inventory records and inventory items.
  11. Submit Button - By clicking this button, the inventory record will be updated with the given information if all inputs are valid.
