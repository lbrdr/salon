1.1 - Login Screen

The login screen is the first screen to be encountered when opening the program. Here, the user can be verified before using the system.

The login screen
  1. Username Field - The user will enter their username here.
  2. Password Field - The user will enter their password here.
  3. Login Button - After filling out the username and password fields, the user can click the login button to verify these credentials. If the verification is successful, they will be logged in in the system.
  4. Forgot Password Link - The user can click this link to proceed to the Forgot Password procedure.
