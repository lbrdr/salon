1.2 - Forgot Password

The Forgot Password is a recovery process for users who may have forgotten their login credentials. Here, the user is able to set a new password for their account after correctly answering a security question, allowing them to enter and use the system.

Link to forgot password
Input for username verification
Input for the security question answer
Input for the new password
  1. Forgot Password Link - The user has to select the forgot password in the login screen to proceed to the Forgot Password screen.
  2. Username Field - The user would have to input their username in this field to check if their account exists.
  3. Security Question and Answer Field - The user has to answer the security question here. The security question and its answer are set upon creating the account.
  4. New Password Fields - The user then creates a new password that would replace the old password that is already forgotten.
