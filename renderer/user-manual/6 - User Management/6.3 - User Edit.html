6.3 - User Edit

The User Edit Page is

The screen
  1. Username Field - The admin will enter the username here to check if it exists.
  2. Full Name Field - The admin will input the full name of the user in this field.
  3. User Type Dropdown - This dropdown allows the admin to select the type of user registered, such as 'Admin' or 'Staff'. This determines the user's access level within the system.
  4. Password Field - The admin can set a new password for the user here. This should follow the system's password strength guidelines.
  5. Confirm Password Field - The admin will re-enter the new password to confirm it.
  6. Login Dropdown - This dropdown is used to select the type of login (e.g., enabled or disabled) for a user.
  7. Security Question Field - The admin can create a security question for the specific user account.
  8. Security Question Answer Field - The admin can enter the answer to the selected security question. This will be used to verify the user's identity if password recovery is needed.
