6.1 - User Management Page

The User Management Page provides a central interface for managing user accounts within the system. Here, administrators can register new users, edit existing user details, and manage user roles and permissions.

The user management screen
  1. Register New User Button - This button allows the administrator to add new users to the system.
  2. Users Table - This table shows the users registered in the system and their details such as their user ID, their username, their full name, their user type, and their login type.
  3. Edit User Button - This button allows the administrator to modify the details of existing users.
