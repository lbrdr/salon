6.2 - User Registration

The User Registration screen allows the admin to create new user accounts for the system. This includes fields for entering user details and setting up login credentials.

The user registration screen
  1. Username Field - The admin will enter the desired username for the new user here. This must be unique for each user.
  2. Full Name Field - The admin will input the full name of the new user in this field. This helps in identifying the user.
  3. User Type Dropdown - This dropdown allows the admin to select the type of user being registered, such as 'Admin' or 'Staff'. This determines the user's access level within the system.
  4. Password Field - The admin will set a password for the new user here. This should follow the system's password strength guidelines.
  5. Confirm Password Field - The admin will re-enter the password to confirm it. This ensures that there are no typing errors in the password.
  6. Login Dropdown - This dropdown is used to select the type of login (e.g., enabled or disabled) for a user.
  7. Security Question Field - The admin will create a security question for the specific user account. This will be used for account recovery purposes.
  8. Security Question Answer Field - The admin will enter the answer to the selected security question. This will be used to verify the user's identity if password recovery is needed.
