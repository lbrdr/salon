9.1 - Help Page

The Help Page provides users with comprehensive guidance on how to use the system, offering detailed instructions for various functionalities through user manual and addressing common queries through frequently asked questions.

The help page screen
  1. User Manual - Clicking this button will provide users detailed instructions on how to navigate and use the system effectively.
  2. Frequently Asked Questions - Clicking this button will provide users quick solutions and clarifications to enhance the user experience.
