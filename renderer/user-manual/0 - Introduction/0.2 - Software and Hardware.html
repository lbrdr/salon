0.2 - Software and Hardware Requirements

The system requires specific hardware and software to ensure optimal performance for staff and admin users.

  1. Hardware - The are the minimum requirements for Hardware, it should have a processor equivalent to at least an Intel Core i3, at least 4 GB of RAM, and a minimum of 128 GB SSD storage. Additional peripherals include a monitor with a resolution of 1920x1080 or higher, a keyboard, a mouse, and backup storage like an external hard drive or network-attached storage.
  2. Software - For software, it should have at least an operating system of Windows 10 or later, and must use the latest version of Chrome, Firefox, or Edge browsers. Additional necessary software includes backup software for regular database and application data backups, security software such as antivirus and firewall, and a PDF reader for documentation and reports.

These requirements ensure smooth and efficient system operation for all users.
