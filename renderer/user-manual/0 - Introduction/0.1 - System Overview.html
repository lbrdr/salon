0.1 - System Overview

This system is made to automate the customer management and sales process for Hairliners Salon, enhancing efficiency and improving customer service.

The system consists of various integrated modules to help fulfill its intended purpose. Here are the modules available in the system:

  1. Login - Login Module ensures safe access and user accountability.
    2. The login screen
  2. Customer Management - The Customer Management displays list of customers as well as enables registration of new customers and quick information retrieval.
    3. The customer management screen
  3. Point of Sales - Point of Sales facilitates list of sales record and facilitates record creation.
    4. The point of sales screen
  4. Inventory Management - Inventory Management displays list of stock as well as managing stock updates and usage tracking.
    5. The inventory screen
  5. User Management (Admin Only) - User Management displays staff members and user type, it also monitors user actions/activity.
    6. The user management screen
  6. Report (Admin Only) - Report generates comprehensive reports on customers, sales, inventory, and user activities.
    7. The report screen
  7. Backup and Restore (Admin Only) - The Backup and Restore allows to backup and restore data.
    8. The backup and restore screen
  8. Help - Help provides a User Manual and FAQ for educating the user on how to user the system.
    9. The help screen
  9. About - About offers information about the software and its developers.
    10. The about screen
