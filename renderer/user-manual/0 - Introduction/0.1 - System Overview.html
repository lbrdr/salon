0.1 - System Overview

This system is made to automate the customer management and sales process for Hairliners Salon, enhancing efficiency and improving customer service.

The system consists of various integrated modules to help fulfill its intended purpose. Here are the modules available in the system:

  1. Login Module - Login Module ensures safe access and user accountability.
    2. The login screen
  2. Customer Management - The Customer Management Module displays list of customers as well as enables registration of new customers and quick information retrieval.
    3. The customer management screen
  3. Point of Sales - Point of Sales Module facilitates list of sales record and facilitates record creation.
    4. The point of sales screen
  4. Inventory - Inventory Module displays list of stock as well as managing stock updates and usage tracking.
    5. The inventory screen
  5. User Management (Admin Only) - User Management Module displays staff members and user type, it also monitors user actions/activity.
    6. The user management screen
  6. Report (Admin Only) - Report Module generates comprehensive reports on customers, sales, inventory, and user activities.
    7. The report screen
  7. Maintenance (Admin Only) - Maintenance Module allows to backup and restore data.
    8. The maintenance screen
  8. Help - Help Module generates provides a User Manual and FAQ
    9. The help screen
  9. About - About Module offers software and developers information.
    10. The about screen
