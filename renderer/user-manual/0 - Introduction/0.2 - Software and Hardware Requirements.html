0.2 - Software and Hardware Requirements

The Hardware and Software tools used in the development of this project are the following:

  1. Hardware
    To operate the system, the minimum hardware requirements are:
    • a processor equivalent to at least an Intel Core i3,
    • at least 4 GB of RAM, and
    • a minimum of 128 GB SSD storage.

    Additional peripherals include:
    • a monitor with a resolution of 1280x720 or higher,
    • a keyboard,
    • a mouse,
    • and backup storage like an external hard drive or a network-attached storage.

    The development of the system was done in an MSi Bravo 15 C7UDX high-performance laptop, equipped with an AMD Ryzen 5 7535HS processor, 512GB NVME storage and 8GB DDR5-4800 RAM.

  2. Software
    To operate the system, the minimum software requirements:
    • it should have at least Windows 10 or later as the operating system.

    Additional necessary software includes:
    • backup software for regular database and application data backups,
    • security software such as antivirus and firewall,
    • and a PDF reader for documentation and reports.

    The development of the system was done on Windows 11 Home using Notepad v8.6.

These requirements ensure smooth and efficient system operation for all users.
