8.1 - Backup and Restore Page

The Backup and Restore Page allows administrators to manage the system's data backups and restoration processes.

The backup and restore screen
  1. Backup Data Button - Clicking this button initiates the backup process, which creates a copy of all system data to an external location for safekeeping.
  2. Restore Data Button - Clicking this button restores previously backed-up data to the system, overwriting current data with the restored version.
