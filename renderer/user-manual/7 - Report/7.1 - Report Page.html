7.1 - Report Page

The Report Page is exclusive to administrators and allows them to generate and manage various types of reports. Administrators can filter reports by date and type, ensuring they retrieve the necessary information efficiently.

  1. Report Type - This dropdown allows the admin to select the type of report they wish to generate. Options may include customer report, sales report, inventory report, and user report.
  2. Start Date - In this field, the admin specifies the starting date for the report. It allows the admin to filter data from a specific point in time.
  3. End Date - In this field, the admin specifies the ending date for the report. In conjunction with the start date, it defines the time range for the data included in the report.
  4. Customer Type Selection (For customer report) - In this selection, the admin can specify the type of customers to be reported on. Report for newly-registered/non-registered customers will show details about customer registration and customers in sales records within the specified date range.
  5. Total Expenses Field (For sales report) - In this field, the admin can input the total expenses of the salon, and it will be used in computing the net income at the end of the sales report.
  6. User Selection (For user report) - In this selection, the admin can select all users, or a specific user to be reported on.
  7. Generate Report - After selecting the report type and defining the date range, the admin clicks this button to generate the report. The system compiles the data and presents it in the chosen report format.
  8. Print/Save Report - Once the report is generated, the admin can choose to print/save the report. This function ensures that the report can be easily accessed and utilized for record-keeping or further analysis.
