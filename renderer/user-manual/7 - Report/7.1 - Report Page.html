7.1 - Report Page

The Report Page is exclusive to administrators and allows them to generate and manage various types of reports. Administrators can filter reports by date and type, ensuring they retrieve the necessary information efficiently.

The Report Page screens
  1. Report Type - This dropdown allows the admin to select the type of report they wish to generate. Options may include customer report, sales report, inventory report, and user report.
  2. Start Date - In this field, the admin specifies the starting date for the report. It allows the admin to filter data from a specific point in time.
  3. End Date - In this field, the admin specifies the ending date for the report. In conjunction with the start date, it defines the time range for the data included in the report.
  4. Generate Report - After selecting the report type and defining the date range, the admin clicks this button to generate the report. The system compiles the data and presents it in the chosen report format.
  5. Print/Save Report - Once the report is generated, the admin can choose to print/save the report. This function ensures that the report can be easily accessed and utilized for record-keeping or further analysis.
  6. Customer Type - When Customer Report is selected, the admin can select what type of customer they want to include in the report.

  7. Total Expenses Field - When Sales Report is selected, the admin can enter total expenses spent by the business.
